# Getting Started.

This guide will help you understand how to configure the the system.

## How do I set up authentication?

The authentication token is generated by the system and will be sent to the user.
Our API allows you to easily configure your settings. You should probably review
the documentation before proceeding, as it might cause issues if the steps
are not followed in order.

To learn more, [click here](https://example.com).

We believe this feature lets you build better integrations. It enables you to
connect multiple data sources and provides you with a unified view of your
customers across all channels, touchpoints, and interaction points in a way that
makes it possible to understand the full customer journey from initial awareness
through purchase and beyond.
